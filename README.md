##  Project
- build the printer mechanism, the core
- code tec1 to controll tx and rx and printer/display system
  - tx/rx from 
    - MINT keypad 
    - Hex pad 
  - print to 
    - paper tape
    - 7seg
    - LCD matrix
    - small thermal printer  
- backup transmission to audio tape to print later
